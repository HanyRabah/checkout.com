
## Getting Started

You need to make sure you have nodeJs and Yarn installed
then run 

First of All to download the packages

```bash
yarn 
```

To build and run the project localy

```bash
yarn Build
# then
yarn start
# or
yarn dev
```

## Test
there is 3 test files for ( Form, Input and Rating components)

### to run test
```bash
yarn test
```
